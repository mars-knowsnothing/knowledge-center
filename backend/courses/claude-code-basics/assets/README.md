本目录包含 Claude Code 基础教程中使用的所有资源文件，包括图片、文档和示例代码。

### 文件组织结构

```
assets/
├── images/              # 图片资源
│   ├── screenshots/     # 屏幕截图
│   ├── diagrams/        # 架构图和流程图

│   ├── basic/           # 基础示例

```

## 🖼️ 图片资源

所有屏幕截图都经过标准化处理：

## 🔧 使用说明

### 在课程中使用资源

1. **图片引用**: 使用相对路径引用图片

3. **模板文件**: 作为练习的起始点

### 本地开发环境

如果您在本地运行这些示例：

```bash

npm install

# 运行示例项目
npm run dev

# 运行测试
npm test
```
