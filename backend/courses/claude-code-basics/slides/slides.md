# 🎯 有效提示技巧

### 1. 具体描述需求
❌ **模糊：** "帮我写一个函数"
✅ **清晰：** "创建一个 TypeScript 函数，接收用户 ID 数组，返回用户详细信息"

### 2. 提供上下文
```bash
claude> 在这个 Next.js 项目中，
我使用 Tailwind CSS，
需要创建一个响应式的用户卡片组件
```

### 3. 迭代改进
```bash
claude> 这个组件很好，但能否
1. 添加加载状态
2. 支持暗色主题
```
