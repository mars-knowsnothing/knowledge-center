# HTML Basics

HTML (HyperText Markup Language) is the foundation of web pages.

## Key Concepts:
- Elements and tags
- Attributes
- Document structure
- Semantic HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>
```
