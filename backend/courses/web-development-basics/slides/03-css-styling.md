# CSS Styling

CSS (Cascading Style Sheets) controls the presentation of HTML elements.

## CSS Syntax:
```css
selector {
    property: value;
}
```

## Example:
```css
h1 {
    color: blue;
    font-size: 2em;
    text-align: center;
}

.container {
    max-width: 800px;
    margin: 0 auto;
    padding: 20px;
}
```
